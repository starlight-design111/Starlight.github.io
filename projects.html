VRChat Environment: Cyber-Lounge Pipeline Study

Role: 3D Artist & Technical Artist | Tools: Unity 2022, Blender 3.x, VRChat SDK

Proprietary Assets Optimized UVs VR Performance

This project demonstrates a full-pipeline asset workflow, moving from original modeling and UV layout in Blender to technical lighting implementation within Unity 2022 for VRChat.

Part 1: Asset Creation (Blender)

I modeled 100% of the room geometry and props shown below, focusing on clean topology for VR. The UV checker map image proves consistent texture density and optimized UV layouts, which are critical for artifact-free lightmap baking.

Part 2: Implementation & Lighting (Unity 2022)

Once imported to Unity, I designed the lighting system. I utilized Light Probes to bridge the gap between static baked geometry (walls, bed) and dynamic objects (players, interactables), ensuring avatars are lit correctly by the emissive hexagonal panels.

UE5: Line Trace Puzzle Mechanics

Role: Technical Designer & Systems Programmer | Tools: Unreal Engine 5.3, Blueprints

Line Tracing Verification Logic Level Layout

The goal of this project was to develop a "match-key-to-lock" mechanic using colored states. The player must acquire a color variable and interact with corresponding colored doors to progress.

Technical Highlights:

  • Interaction System: Engineered a precise Line Tracing system that detects valid "Interactable" interfaces when the player presses 'E'.
  • Color Matching Logic: Developed the back-end variable check that compares the player's held color state (represented by the UI sphere) against the required door color state. The door only triggers an open animation sequence upon a successful match.
  • Player Feedback Loops: Implemented reactive UI and environmental cues (emissive material shifts) to confirm successful or failed interactions.
